# Sql Table Formatter
A tiny Windows console app that converts pasted tab-separated data into a rendered HTML table and opens it in your browser so you can copy & paste it directly into emails (Outlook, Gmail, Teams, etc).

---
### What it does
* You paste your tab-separated data into the console
* The app renders it as a real HTML table
* Opens it in your default browser
* You copy the table and paste into your email

### Self-publishing
1. Download the source code
2. Navigate to project folder
3. Run the folowing: `dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true`
